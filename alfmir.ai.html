Alfmir.ai is an open-source, MIT-licensed AI software project focused on agentic coding workflows, project management, image design, and AI chat with embedded web search.

The project is currently in demo / open beta form. Production accounts, billing, and full commercial infrastructure are not yet launched. There are currently no paid customers or active subscriptions.

The long-term goal is to offer Alfmir.ai as a real SaaS product with planned free and paid tiers, while keeping the core project open source and self-hostable.

Open Beta includes limited access to an open-model agent, Qwen Coder 30B A3B, subject to capacity.

Premium model execution is planned through credit-based usage for frontier coding models such as KwaiPilot: KAT Coder Pro V2.

Engineering work

Alfmir.ai has been developed over multiple years as an open-source software project and professional portfolio project, with recent work focused on making the demo more complete and production-ready.

Responsibilities include product strategy, full-stack development, cloud/DevOps, data pipelines, and integrations across AI APIs, project workflows, search, design tools, and commerce experiments.

Architected and implemented Alfmir.ai services in Node.js and Python (github.com/alfmir-ai):

  • AI software development automation tooling
  • Project/task management workflows
  • AI chat with embedded web search
  • Image design workflows
  • Finance dashboards and accounting utilities
  • Marketplace transaction reconciliation experiments for eBay and Amazon
  • Printify integration experiments for on-demand apparel workflows

Commerce operations

Lochner Apparel is an active AI-assisted apparel project using generated designs and on-demand fulfillment concepts through Printify and marketplace storefronts, currently operating at a small scale (about 4 sales per month).

The apparel work is separate from Alfmir.ai's core code-agent product, and continues to support design generation, marketplace workflows, product publishing, and transaction reconciliation.
